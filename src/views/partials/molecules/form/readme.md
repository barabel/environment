# компонент form

## Пропсы:
 - className: string, дополнительные классы;
 - props: список  пропсов для рендера компонента
 - action: string, api Формы

## модификаторы
- form--horizontal - делает раскладку формы горизонтальной, начиная от разрешения 1280px.
